# 🤖 Blenderbot Chat

👉 Available at: https://blenderbot.matx.dev

### 🔨 Tech

- Go 1.19
- Html, CSS, JS
- Blenderbot Hugging Face Inference API

### 🏠 Run it locally

```bash
go run .
```

with docker:

```bash
docker build -t blenderbot-chat . &&
docker run -p 3000:3000 blenderbot-chat
```

OR use **[air](https://github.com/cosmtrek/air)** instead.
